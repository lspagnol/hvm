# HVM
**HVM** is a tool designed for two KVM/libvirt/ZFS/Ubuntu hypervisors in ACTIVE/PASSIVE mode

**HVM** est un outil destiné à gérer automatiquement deux hyperviseurs KVM/Libvirt/ZFS/Ubuntu en mode ACTIF/PASSIF
